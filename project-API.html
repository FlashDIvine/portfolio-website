Project 3 - API Toko Online

RESTful API berbasis Laravel untuk manajemen produk dengan sistem autentikasi Bearer Token. API ini mendukung fitur CRUD produk serta upload gambar menggunakan form-data.

Teknologi

Laravel MySQL Postman REST API

Fitur Utama

  • Login & autentikasi menggunakan Bearer Token
  • Proteksi endpoint (401 Unauthorized)
  • CRUD produk (Create, Read, Update, Delete)
  • Upload gambar produk

Autentikasi

Endpoint dilindungi dan membutuhkan token untuk diakses.

Login untuk mendapatkan token:

Gunakan token pada header Authorization untuk akses endpoint.

Endpoint

POST /api/login

GET /api/products

POST /api/products

DELETE /api/products/{id}
