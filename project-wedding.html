Website Invitation Wedding

Website undangan pernikahan berbasis web yang memungkinkan tamu melihat informasi acara, melakukan RSVP, menerima email konfirmasi otomatis, serta mendapatkan QR code sebagai tiket digital.

Preview Website

Teknologi

Laravel Blade MySQL Laravel Mail QR Code Generator

Fitur Utama

  • Halaman informasi pernikahan
  • RSVP (konfirmasi kehadiran tamu)
  • Pengiriman email konfirmasi otomatis
  • Pembuatan QR code sebagai tiket tamu
  • Validasi tamu menggunakan QR code saat acara
  • Desain responsif (mobile-friendly)
  • Manajemen data tamu (backend)

Cara Kerja Sistem

Tamu mengakses website undangan dan mengisi form RSVP untuk mengonfirmasi kehadiran. Data yang diinput akan disimpan ke dalam database.

Setelah RSVP berhasil, sistem secara otomatis mengirimkan email konfirmasi kepada tamu yang berisi detail acara serta QR code unik sebagai tiket masuk.

QR code tersebut digunakan saat hari acara sebagai validasi kehadiran tamu, sehingga proses check-in menjadi lebih cepat, praktis, dan terorganisir.
